# Лесная Обитель · Style Guide

Премиальная экотуристическая база отдыха в Крыму. Бренд сочетает три смысловых
слоя: **лес + активный отдых + винная культура**.

> «Тишина, которую не привозят.»

---

## 0. Принципы

1. **Связь с природой** — пастельные лесные тона, тёплые оттенки земли, никакого
   неонового и кислотного.
2. **Активный отдых** — структурно ясные сетки, чёткая типографика, удобные
   тачабельные элементы.
3. **Винная культура** — золотые и розовые акценты, курсивная антиква
   Cormorant Garamond для цитат и описаний.
4. **Доверие** — щедрые поля, мягкие тени, никаких агрессивных кампейн-приёмов.
5. **Доступность по умолчанию** — WCAG 2.1 AA минимум, focus-видимый, touch ≥ 44px.

## 6. Motion

```
--t-fast:    150ms ease-in-out;     /* state changes */
--t-base:    300ms ease-in-out;     /* hover, transitions */
--t-slow:    500ms ease-in-out;     /* секционные fade */
--t-spring:  400ms cubic-bezier(0.34, 1.56, 0.64, 1);  /* «живые» элементы */
--t-natural: cubic-bezier(0.22, 1, 0.36, 1);
```

**Reduced motion** учитывается через `@media (prefers-reduced-motion: reduce)` —
все анимации сводятся к 0.01ms.

## 9. Формы

- **Input**: `min-h 44 px`, border 1.5 px `--c-surface`, focus border `--c-moss`
  + ring 3 px `--sh-focus`.
- **Placeholder**: `--c-ink-mute`.
- **Error**: border `--c-error`, иконка слева, microcopy под полем.
- **Success**: toast slide-in справа, авто-скрытие 4 с.

## 11. UX-копи

### Tone of voice

- **Дружелюбный** — «вы» c прописной не используем; разговорно, без канцелярита.
- **Экспертный** — конкретные цифры, спецификации байков и вин — там, где нужно.
- **Расслабленный** — никакого давления, никаких «купи сейчас!».

### Кнопки

| Контекст                     | Текст                                |
|------------------------------|--------------------------------------|
| Главный CTA                  | Забронировать                        |
| Карточка — детали            | Узнать подробнее                     |
| Карточка — действие          | Выбрать и забронировать              |
| Форма — отправка             | Отправить заявку                     |
| Форма — загрузка             | Отправляем…                          |
| Подтверждение                | Готово. Свяжемся в течение часа.     |
| Маршруты                     | Посмотреть маршрут                   |

### Microcopy

| Поле                | Placeholder / hint                                      |
|---------------------|---------------------------------------------------------|
| Имя                 | Как вас зовут?                                          |

## 12. Доступность (WCAG 2.1 AA)

- [x] Контраст ≥ 4.5:1 для body / ≥ 3:1 для крупного текста
- [x] `:focus-visible` со сменой outline/shadow
- [x] Touch-target ≥ 44 × 44 px
- [x] Семантические теги: `header / main / nav / section / footer`
- [x] Заголовки иерархично: h1 → h2 → h3
- [x] `aria-label` на иконочных кнопках; `aria-hidden="true"` на декоративных иконках
- [x] `<label>` для каждого поля + `aria-describedby` для подсказок
- [x] `prefers-reduced-motion` — анимации отключаются
